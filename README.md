# AI ChatBot

A machine-learning chatbot built using Python, NLTK, and scikit-learn.

## Features

- Intent classification
- NLP preprocessing
- TF-IDF vectorization
- Logistic Regression
- Conversation memory
- Confidence scoring
- Model persistence

## Training

Run "python train.py" to train models

## Chat

Run "python chat.py" to run models

##Menu:
Type these Commands:
  help    - Show help menu
  history - Show full chat log
  clear   - Wipe the session memory
  stats   - Show total active turns
  quit    - Exit the chat
